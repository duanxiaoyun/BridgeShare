Android Icon Guidelines Template Pack, version 4.0
October 2011


SUMMARY

    This Android Icon Templates Pack contains a series of image templates that
    you can use to get started designing custom icons for your Android
    applications. It includes templates for a variety of icon types, styles, and
    sizes. To use the templates, load them in your image editor and edit to suit
    your needs. Make sure that your icons follow the general guidelines
    described in the Icons Design Guidelines document, which is available at:

    http://developer.android.com/guide/practices/ui_guidelines/icon_design.html

COMPATIBILITY

    These templates are designed for compatibiility with Android versions 4.0
    and earlier.

MORE INFORMATION

    For general information about how to develop Android applications, including
    how to implement icons, read the Developer's Guide. 

QUESTIONS AND FEEDBACK
